# GitHub Enterprise - Administration

## What "Administration" Means

Administration is about **setting up and running your organization’s GitHub space** —  
like managing a digital office building where teams store and work on projects.  

------------------------------------------------------------------------

### Core Ideas

|        Core Idea | What it is                          | Think of it like         |
| ---------------: | :---------------------------------- | :----------------------- |
| **Organization** | Your company’s main workspace       | The office building      |
|   **Repository** | Where a project’s files live        | A shared folder          |
|         **Team** | A group of people who work together | A department             |
|       **Member** | A person with access                | An employee with a badge |

------------------------------------------------------------------------

### Identity & Access

|             Concept | What it is                    | Think of it like             |
| ------------------: | :---------------------------- | :--------------------------- |
|  **Authentication** | Proving who you are           | Showing your ID              |
|  **Single Sign-On** | One login for everything      | One master key for all doors |
| **Two-Factor Auth** | Extra login security          | Lock + phone code            |
|     **Permissions** | What someone is allowed to do | View, edit, or manage rights |

------------------------------------------------------------------------

### Roles

|       Role | What it is                  | Think of it like      |
| ---------: | :-------------------------- | :-------------------- |
|  **Owner** | Full control of everything  | The building manager  |
|  **Admin** | Manages settings and people | The office supervisor |
| **Member** | Regular user                | A staff member        |

------------------------------------------------------------------------

### Governance & Safety

|        Concept | What it is                       | Think of it like             |
| -------------: | :------------------------------- | :--------------------------- |
|   **Policies** | Rules for how things should work | The employee handbook        |
|  **Audit Log** | Record of important actions      | Security cameras for changes |
| **Compliance** | Following laws and company rules | Passing inspections          |

------------------------------------------------------------------------

### Licensing & Billing

|        Term | What it is                      | Think of it like |
| ----------: | :------------------------------ | :--------------- |
| **License** | Permission to use GitHub        | A subscription   |
|   **Seats** | Number of people who can use it | Number of desks  |

------------------------------------------------------------------------

### Everyday Summary

Administration is about **who gets in, what they can do, and keeping everything safe  
and organized** — just like running a well-managed office.
