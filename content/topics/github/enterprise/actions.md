# GitHub Enterprise - Actions

## Actions

Actions are **automatic helpers** that do work for you---like digital
assistants that run tasks whenever something happens.

|    Core Idea | What it is                           | Think of it like                     |
| -----------: | :----------------------------------- | :----------------------------------- |
| **Workflow** | A set of instructions for automation | A to-do list that runs by itself     |
|  **Trigger** | It starts the automation             | A motion sensor that turns on lights |
|      **Job** | A big task in a workflow             | A work shift                         |
|     **Step** | A small task inside a job            | One item on a checklist              |

### Why Actions Matter

|       Core Idea | What it is                           | Think of it like                         |
| --------------: | :----------------------------------- | :--------------------------------------- |
|  **Automation** | Letting computers do repetitive work | A coffee machine instead of hand brewing |
| **Consistency** | Doing things the same way every time | Following a recipe                       |
|       **Speed** | Getting results faster               | Using the express checkout lane          |

### Common Uses

|      Use Case | What it is                         | Think of it like                     |
| ------------: | :--------------------------------- | :----------------------------------- |
|   **Testing** | Checking if work is correct        | Proofreading before printing         |
|  **Building** | Turning work into something usable | Baking ingredients into a cake       |
| **Deploying** | Making work available to people    | Putting the cake on the table to eat |

### Safety & Control

|     Safety Tool | What it is                         | Think of it like                       |
| --------------: | :--------------------------------- | :------------------------------------- |
|     **Secrets** | Hidden info like passwords or keys | A locked drawer for valuables          |
| **Permissions** | What Actions are allowed to do     | Giving a helper a limited-access badge |

### Everyday Summary

Actions are **automatic workers** that save time, reduce mistakes, and
keep work running smoothly.
