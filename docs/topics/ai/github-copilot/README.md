# GitHub Copilot

## ✨ Say Hello to GitHub Copilot Enterprise
Using Copilot in GitHub Enterprise to do everyday work  
> Coding is more than just writing code. It's about understanding where to place  
> your code and how to write it effectively. GitHub Copilot Enterprise is here to  
> support you whenever you are in your project! With the ability to ask questions  
> in natural language about your project and the frameworks and languages you're  
> using, Copilot helps you navigate your coding challenges.  
>   
> Use code completion and chat to refine your code, and let Copilot automatically  
> generate your pull request summaries. Explore the full potential of these features.

------------------------------------------------------------------------

## Issues (Web)  
**Solving issues with natural language**

Copilot helps you turn everyday language into clear, actionable work items.  
You can describe a problem in plain English, and Copilot will help rewrite it
into a clean, understandable issue description.

**What it helps with**
- Writing better issue descriptions  
- Turning rough notes into clear tasks  
- Suggesting acceptance criteria  

**Think of it like:**  
A smart assistant who helps you write clear work tickets.

------------------------------------------------------------------------

## Explain (Web)

Copilot can explain code, files, or changes in simple language — even for people
who don’t normally read code.

**What it helps with**
- Understanding what a file does  
- Explaining technical changes to non-technical teammates  
- Learning unfamiliar projects faster  

**Think of it like:**  
Having a translator that turns “developer talk” into plain English.

------------------------------------------------------------------------

## Suggest Improvements (Web)

Copilot reviews code and suggests better ways to write it — for clarity, safety,
or performance.

**What it helps with**
- Cleaner and easier-to-read code  
- Spotting potential mistakes  
- Encouraging best practices  

**Think of it like:**  
A helpful editor who proofreads your work before it’s published.

------------------------------------------------------------------------

## Implementing Functionality (Web)

You can describe what you want in normal language and Copilot will suggest code
to make it happen.

**What it helps with**
- Turning ideas into working features  
- Reducing time spent on boilerplate work  
- Helping new developers get started  

**Think of it like:**  
Explaining a task to a coworker and having them draft the first version for you.

------------------------------------------------------------------------

## Knowledge Bases (Web)

Copilot Enterprise can use your organization’s documentation, standards, and
guidelines as a **knowledge base**.

**What it helps with**
- Answering questions using your company’s rules  
- Keeping answers consistent with internal policies  
- Reducing repeated “where is that documented?” questions  

**Think of it like:**  
A librarian who knows exactly where your company keeps its answers.

### Immersive Mode (Web)

Immersive mode gives you a focused space to ask deeper questions and explore
topics without distractions.

**Think of it like:**  
Going into a quiet study room instead of working in a noisy office.

### Ensure Accessibility (Web)

Copilot helps check if your work is usable by everyone — including people with
disabilities.

**What it helps with**
- Spotting missing labels or descriptions  
- Encouraging inclusive design  
- Improving usability  

**Think of it like:**  
A friendly reviewer making sure everyone can use what you build.

### References to KB (Web)

Copilot can point back to the documents it used so you can verify information.

**Think of it like:**  
Footnotes in a report — showing where the facts came from.

------------------------------------------------------------------------

## Highlight Class and Ask (VS Code)

You can select a piece of code and ask Copilot questions about it directly.

**What it helps with**
- “What does this do?”  
- “Is this safe?”  
- “Can this be simpler?”  

**Think of it like:**  
Pointing at something and asking a coworker to explain it.

------------------------------------------------------------------------

## Inline Suggestions (VS Code)

Copilot suggests code as you type — finishing lines or even whole sections.

**What it helps with**
- Writing faster  
- Avoiding common mistakes  
- Staying focused on ideas instead of syntax  

**Think of it like:**  
Auto-complete on steroids.

------------------------------------------------------------------------

## Commit Messages (VS Code – Terminal)  
**Generate a commit message**

Copilot can look at what you changed and write a clear summary for you.

**What it helps with**
- Better documentation of changes  
- Consistent wording  
- Saving time  

**Think of it like:**  
A secretary who writes meeting notes for you.

------------------------------------------------------------------------

## Pull Request (Web)

Copilot helps explain and summarize your work when sharing it with others.

### Summary (Web)

Automatically creates a clear description of what your changes do.

**What it helps with**
- Faster reviews  
- Fewer misunderstandings  
- Better project history  

**Think of it like:**  
The back cover of a book — explaining what’s inside.

### Files Changed – Explain (Web)

Copilot explains each changed file in simple terms.

**What it helps with**
- Non-technical reviewers  
- Managers who want the big picture  
- New team members learning the project  

**Think of it like:**  
A tour guide walking you through what changed and why.

------------------------------------------------------------------------

## Big Picture

GitHub Copilot Enterprise is not just for developers — it’s a **productivity
partner** that helps everyone:

- Understand work faster  
- Communicate more clearly  
- Reduce mistakes  
- Save time on repetitive tasks  

**In simple terms:**  
Copilot is like having a knowledgeable teammate who’s always available, never
judges, and loves explaining things clearly.
